# CODSOFT Internship Tasks

This repository contains all the tasks completed as part of my CodSoft Java Programming Internship.

## 🔹 Task 1: Number Guessing Game
A console-based Java application where the user guesses a randomly generated number.

### 🎯 Features:
- Random number generation (1–100)
- Limited number of attempts
- Feedback: Too High / Too Low / Correct
- Multiple rounds using do-while loop
- Score calculation based on attempts

### 💡 Concepts Used:
- Java Basics
- Conditional Statements
- Loops (while, do-while)
- Random class
- Scanner class (user input)

### ▶️ How to Run:
1. Compile the program:
   javac Numbergame.java
2. Run the program:
   java Numbergame
